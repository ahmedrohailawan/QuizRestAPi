# QuizRestAPi
### Requirements
You should have Node Server, And dependencies installed using npm pakage

### How to run
You have to install dependencies using this command ** npm i **
<img src="https://github.com/ahmedrohailawan/QuizRestAPi/blob/main/readme%20files/1.jpeg" width="900">
Then you have to run this command to run server ** nodemon run server  **
<img src="https://github.com/ahmedrohailawan/QuizRestAPi/blob/main/readme%20files/2.jpeg" width="900">

### Dashboard (success response code = 0)
[url : http://localhost:3000/](http://localhost:3000/)
